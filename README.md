# Musiq

## Project Description
Musiq is a modern music website built with Vite and React. It features video content, a merchandise store, and donation support for fans to contribute to your music and ministry.

## Features
- **Featured Videos** - Showcase your music videos, live sessions, and behind-the-scenes content
- **Merchandise Store** - Sell branded products directly to your fans
- **Donation Support** - Accept one-time gifts and monthly contributions from supporters
- **Mobile-First Design** - Fully responsive and optimized for all devices
- **Modern UI** - Built with Tailwind CSS for a professional, cinematic look

## Tech Stack
- **Frontend Framework**: React 18
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS
- **Package Manager**: npm
